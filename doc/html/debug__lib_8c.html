◆ initRetargetSwo()

void initRetargetSwo ( void  )

Initialize the SWD/TPI/ITM debug modules to output debug messages via the Debug interface. This will enable the outputz of printf() and fputc() like messages.

In order to enable the output add the following command at the GDB prompt (not needed when using the GNU ARM Plugin from within Eclipse)

(gdb) monitor SWO EnableTarget 16000000 0 1 0

Open up a separate terminal and connect to the TERMINAL OUTPUT CHANNEL of the Segger JLink.

$ telnet localhost 2333
